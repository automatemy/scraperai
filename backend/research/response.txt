Here is a Python code using BeautifulSoup to extract the required data:

```python
from bs4 import BeautifulSoup

html_doc = """Your HTML string here"""

soup = BeautifulSoup(html_doc, 'html.parser')

# Find all company cards
company_cards = soup.find_all('a', class_='card company-card card_navigator w-inline-block')

# Loop through each company card and extract data
for card in company_cards:
    # Extract name
    name = card.find('h3', class_='company-card__heading pr-150').text
    # Extract description
    description = card.find('p', class_='company-card__description').text
    # Extract price
    price = card.find('div', class_='company-card__revenue')
    if price is not None:
        price = price.text.strip()
    else:
        price = 'Not available'
    # Extract image
    image = card.find('div', class_='company-card__marker')
    if image is not None:
        image = image['style'].split('(')[1].split(')')[0]
    else:
        image = 'Not available'
    
    print(f'Name: {name}\nDescription: {description}\nPrice: {price}\nImage: {image}\n---')
```

Please replace `"Your HTML string here"` with your actual HTML string. This code will print the name, description, price, and image for each company card in the HTML. If the price or image is not available, it will print 'Not available'.
